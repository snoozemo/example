# A Simple React Template

### If your `git` `nodejs` `pnpm` is installed

```bash
# start

git clone https://github.com/snoozemo/react-example.git dirName

cd dirName

pnpm i

pnpm dev

# build

pnpm build
```

### used some tools.

- tailwindcss
- react router
- redux
- axios
- framer-motion
- nprogress
